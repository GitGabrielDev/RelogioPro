## ✨ Sobre o Projeto

O **Relógio Pro** é um aplicativo desenvolvido para oferecer ferramentas essenciais de controle de tempo em uma interface simples, rápida e agradável.

Com ele você pode acompanhar o horário atual, medir intervalos com precisão usando o cronômetro e configurar contagens regressivas através do timer.

---

## 🚀 Funcionalidades

### 🕒 Relógio
- Exibição do horário em tempo real
- Interface limpa e organizada
- Atualização automática

### ⏱️ Cronômetro
- Medição precisa de tempo
- Iniciar, pausar e reiniciar
- Contagem em tempo real

### ⏲️ Timer
- Configuração personalizada
- Contagem regressiva
- Fácil de usar
